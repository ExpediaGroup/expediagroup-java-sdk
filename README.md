# Open World <sup>TM</sup> SDK for Java - Core
The core component of the Open World<sup>TM</sup> SDK for Java.

## Building From Source

1. Build the Project:
    ```
    ./mvnw clean install
    ```

2. (Optional) Generate Documentation:
   ```
   ./mvnw dokka:dokka
   ```
